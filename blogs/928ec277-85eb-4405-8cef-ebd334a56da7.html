The Forward-Forward Algorithm: Some Preliminary Investigations

Created Date: October 21, 2023

This lists a brief summary of the paper.

Rationale: Limitation of back prop

  • Cannot accommodate non-differentiable loss (human preference)
  • precise details of the forward computation need to be known

As a temporary solution, reinforcement learning is used but problems are:

  • suffer from high variance
  • learning rate needs to be inversely proportional to the number of variables that are being perturbed

one with positive (i.e. real) data and the other with negative data

layer-wise goodness function (objective function)

  • sum squared activities of the hidden units
  • above threshold for real data
  • below threshold for negative data

Benefits:

  • Doesn’t use back propagation
  • Use much less memory

Cons:

  • somewhat slower than backpropagation
